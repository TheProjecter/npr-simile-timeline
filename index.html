NPR / Timeline Mashup (Beta)

This NPR / Timeline Mashup is an example of what can be done using the new NPR API.

  • View a Sample Timeline.
  • View the Latest NPR SIMILE Timeline (this particular sample is constantly being updated... at any given point it may return errors or alerts).
  • Put a timeline on your site
    (Note: These scripts are in their early stages and are certain to change).

This script owes an incredible amount to the NPR Digital Media team. You can find out more about the NPR API here. You can also read about what they're up to on their blog Inside NPR.org
